# 使用的工具和库

Serenity 平台使用了一些很有用的开源工具和库，列出如下（按字母顺序排列）：

>此列表可能看起来有点长，但一个 Serenity 的应用程序并没有依赖所有的库。

>其中一些库只在 Serenity 平台自身的发展过程中才用到，而有一些是可选功能的依赖项。

>我们尽量使用开源库，因为它们的优质可以避免重新造轮子。
